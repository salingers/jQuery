Move the mouse and observe the trail smoothness and the Update Time shown below.
Each point in the mouse trail represents one mousemove event.
The trail "cleanup" runs on a setInterval() timer. If the cleanup falls far behind it means the timer is not being called often enough, probably because the other JavaScript code takes too long.
Try both versions of jQuery and each of the selector options.
Especially note performance in IE7 and IE8.
Bonus fun: Overlap a few windows from different browsers and move the mouse around on all of them.
